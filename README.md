# 📌 Loadscreen (MTA:SA)

Sistema de **Loadscreen personalizada** para **Multi Theft Auto (MTA:SA)** com interface moderna, barra de progresso, exibição de informações do servidor e animações durante o carregamento.  

## 🚀 Funcionalidades
- Tela de carregamento totalmente customizada (DX)
- Barra de progresso dinâmica
- Música de fundo

---

## ⚙️ Instalação

1. Copie a pasta **mta_loadscreen** para `server/mods/deathmatch/resources/`.  

2. Certifique-se de adicionar o resource ao `mtaserver.conf`:
```xml
<resource src="mta_loadscreen" startup="1" protected="0" />
```
